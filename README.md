## 사용 기술

- Next.js : SPA 페이지로 제작되었습니다. (app router 형식 적용)
- Styled-Components: css in js 스타일을 적용했습니다.
- Typescript : 타입안정성을 위해 사용했습니다.

## Getting Started

First, run the development server:

```bash
npm run dev
```
